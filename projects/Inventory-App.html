Overview

The ACME Food Bank Inventory System is a comprehensive solution designed to streamline donation tracking, ensure timely usage of perishable items, and generate detailed reports. Built with the MERN stack (MongoDB, Express, React, Node.js), it provides a secure and efficient platform for managing food bank operations.

Problem Statement

Food banks often struggle with inventory management, particularly tracking perishable items and their expiration dates. Manual processes lead to food waste, inefficient distribution, and challenges in reporting to stakeholders. The ACME Food Bank needed a system to efficiently track donations, monitor expiration dates, and provide role-based access for different types of users.

Key Features

Role-Based Access Control (RBAC)

The system implements comprehensive RBAC with distinct roles including Admin, Employee, Volunteer, User, and Donor. Each role has specific permissions tailored to their responsibilities within the food bank operation, ensuring secure and appropriate access to features.

Donation & Inventory Management

The application provides robust features for tracking donations and managing inventory. Users can record new donations, categorize items by type (perishable, non-perishable), assign storage locations, and track donor information. The inventory view offers advanced sorting and filtering capabilities to easily find and manage items.

Expiration Tracking & Reporting

The system highlights items nearing expiration to prevent food waste. Comprehensive reporting tools allow staff to generate monthly inventory reports, track items expiring within specified timeframes, and export data to CSV for further analysis. This ensures efficient use of resources and aids in planning and decision-making.

Enhanced Security with Two-Factor Authentication

To ensure the highest level of security for sensitive food bank data, the system implements two-factor authentication for admin and employee accounts. This additional layer of protection helps prevent unauthorized access and safeguards donor information and inventory records.

Technologies Used

MongoDB NoSQL database
Express.js Backend framework
React Frontend library
Node.js JavaScript runtime
JWT Authentication
Prisma ORM for database

Development Approach

The application was developed using an Agile methodology with sprints focused on incremental feature implementation. The MERN stack was chosen for its flexibility, performance, and scalability in web applications.

MongoDB provides a flexible schema for varied inventory items, Express.js offers a robust API framework, React delivers a responsive and interactive user interface, and Node.js ensures efficient server-side processing. The project follows a modular architecture with clear separation of concerns between frontend and backend components.

Challenges & Solutions

Role-Based Security: Implementing a comprehensive RBAC system was challenging. The solution involved creating a hierarchical role structure with well-defined permissions that controlled access at both the frontend and backend levels.

Expiration Management: Tracking perishable items effectively required designing a system that could filter, sort, and highlight items based on expiration status. Custom date handling and visual indicators were implemented to make expiring items immediately obvious.

User Authentication: Securing the system while maintaining usability was addressed by implementing JWT authentication with two-factor authentication (2FA) support, providing a balance between security and convenience.

Outcome

The ACME Food Bank Inventory System successfully addresses the challenges of food bank inventory management with a secure, user-friendly solution. Key outcomes include:

  • Streamlined donation processing and inventory tracking
  • Reduced food waste through effective expiration monitoring
  • Enhanced reporting capabilities for better decision-making
  • Improved security through role-based access control
  • User-friendly interface accessible on mobile and desktop devices

The application demonstrates how the MERN stack can be leveraged to create efficient, secure inventory management systems for organizations with complex tracking needs.
