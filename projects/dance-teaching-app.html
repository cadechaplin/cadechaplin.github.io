Dance Teaching App

Python Google ML UI/UX
View on GitHub

Overview

The Dance Teaching App is an AI-powered application that uses Google ML Pose Detection to help users learn dance moves and receive real-time feedback on their performance. This project was developed as part of the WSU CrimsonCode hackathon in 2025, where it won first place.

Problem Statement

Learning dance moves traditionally requires in-person instruction or following along with videos without any feedback. This makes it difficult for beginners to know if they're performing the moves correctly.

Solution

Our application uses computer vision and machine learning to:

  • Detect the user's body pose and movements in real-time
  • Compare the user's movements to reference dance moves
  • Provide instant feedback on accuracy and suggestions for improvement
  • Track progress over time and offer personalized practice recommendations

Technologies Used

Python Core programming language
Google ML Kit For pose detection and tracking
OpenCV For camera input and image processing
PyQt For building the user interface

My Role

As the lead developer on this project, I was responsible for:

  • Implementing the pose detection algorithm integration
  • Designing and developing the user interface
  • Creating the comparison algorithm between reference poses and user poses
  • Training the system on various dance moves

Challenges & Solutions

One of the biggest challenges was accurately detecting subtle movements in dance routines. We solved this by implementing a custom algorithm that focuses on key joint relationships rather than absolute positions, making the system more robust.

Outcome

The Dance Teaching App won first place at the WSU CrimsonCode hackathon. Judges were impressed by the practical application of machine learning technology and the potential impact on remote learning for physical activities.
